SWEEPY (Still in early development)

Sweepy is a cute little disk analyzer tool that helps you figure out what’s taking up space on your computer.
Ever wondered why you only have 25 KB of storage left? Sweepy helps you find:
Duplicate files
Junk/system clutter
Unused files

Current Status:
Sweepy is still in very early development, so features are limited right now — but a lot more is planned!

Planned Features:
Multiple storage visualization modes
Custom themes, colors, and styles
A beautiful modern UI with glowing visuals and buttery-smooth 60+ FPS animations
Sweepy himself — a cute little mascot dedicated to cleaning your computer, organizing your disk or plotting world domination
Optional sound effects and ambience
More optimization and cleanup tools
And much more!

Thanks so much for checking out Sweepy!
